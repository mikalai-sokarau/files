- Что такое AngularJS?
- AngularJS это MVC фреймворк для создания SPA приложений.

- Что значит MVC?
- MVC расшифровывается как Model (отвечает за управление данными), View (отвечает за 
пользовательский интерфейс), Controller (отвечает за бизнес-логику).

- Чем AngularJS выражения отличаются от JavaScript выражений?
- AngularJS выражения не поддерживают условия, циклы и исключения.
AngularJS выражения поддерживают фильтры.

----------
ng-bind = {{}}, ng-model,
----------

- Что нужно сделать, чтобы запустить AngularJS приложение?
- Необходимо добавить атрибут ng-app="appName" в элемент, который будет служить 
корнем приложения, затем создать новый модуль angular.module("appName", []).
Массив вторым аргументом служит для создания нового модуля, если передавать только
один аргумент AngularJS будет искать модуль созданный ранее. Ng-app автоматически
инициализирует приложение когда оно загружается.

- Что вы знаете про $scope?
- $scope выполняет роль модели в приложении и служит связующим звеном между контроллером
и представлением. Обычно он содержит данные или методы для взаимодействия с view.
Так же в приложении имеется один объект $rootScope, он является родителем для всех 
остальных $scope.

- Что такое контроллер?
- Контроллер это функция, которая обычно принимает в аргументах пустой объект $scope и 
добавляет в него поля и функции, которые будут использоваться далее во view.

- Что такое директивы?
- Директивы позволяют расширять стандартную разметку HTML и описывать поведение отдельных
элементов.

- Что такое сервисы?
- Сервисы являются специальными объектами или функциями, которые выполняют общие для всего
приложения задачи. В AngularJS существуют встроенные сервисы, такие как $q, $http и т.д., а
так же существует возможность создавать свои собственные.

- Как настроить обмен данными между двумя директивами?
- Есть несколько вариантов настроить общение между директивами:
  # Сервисы. Этот вариант является предпочтительным и наиболее часто используемым.
  # События (events). 
    $emit отправляет событие вверх по иерархии. 
    $broadcast вниз.
    $on позволяет подписаться на событие.
  # Добавлять данные в глобальный объект $rootScope.
  # Через общий родительский элемент.

- Что вы можете сказать о binding types в AngularJS?
  @:  text binding 
  =:  two-way binding
  &:  method binding
  <:  one-way binding

- Как вы понимаете выражение Dependency Injection?
- DI это паттерн / техника, внедрения внешних зависимостей, при котором между компонентами 
отсутствуют жесткие связи. Обычно внедряемая зависимость передается в аргументах и далее 
используется в бизнес логике компонента.

- Что вы знаете о жизненном цикле директивы?
- Жизненный цикл директивы состоит из фаз:
  -> compile. После загрузки HTML разметки браузером AngularJS вызывает функцию compile 
  у всех обнаруженных директив.
  -> controller. На этом этапе AngularJS создает контроллер для директивы.
  -> pre-link
  -> post-link
